# 🚀 RetailIQ — Sales Intelligence Platform

A modern, interactive **Retail Analytics Dashboard** designed to streamline sales tracking, monitor performance, and generate actionable insights for retail businesses.

Built with a focus on **clean UI, real-time insights, and usability**, RetailIQ helps store managers and analysts make data-driven decisions efficiently.

---

## 📌 Project Overview

RetailIQ is a **web-based sales intelligence system** that allows users to:

- Record and manage sales transactions  
- Monitor KPIs like revenue, transactions, and performance  
- Visualize data through interactive charts  
- Analyze store, brand, and category-level performance  
- Export and manage data seamlessly  

This project also supports integration with a **SQL-based database system** for scalable and structured data storage.

---

## ✨ Key Features

### 📊 Analytics Dashboard
- Total Revenue, MTD, YTD metrics
- Transactions & Units Sold tracking
- Best & Worst Store insights
- Category & Brand performance analysis
- Interactive charts using Chart.js:
  - Revenue Trend 📈
  - Sales by Brand 📊
  - Sales by Category 📉
  - Product Distribution (Donut Chart)

---

### 🧾 Sales Entry System
- Dynamic product selection:
  - Category, Sub-category
  - Size, Color, Fit, Season
- Real-time **Bill Preview**
- Discount calculation
- Automatic revenue updates
- Data sync with SQL database / Google Sheets

---

### 🔐 Secure PIN-Based Login
- Store-specific 4-digit PIN system
- Lockout protection after failed attempts
- Visual PIN entry system with animations
